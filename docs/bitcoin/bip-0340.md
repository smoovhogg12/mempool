# BIP-340 — Schnorr Signatures for secp256k1

## Summary

[BIP-340](https://github.com/bitcoin/bips/blob/master/bip-0340.mediawiki) defines a standard for Schnorr signatures over the secp256k1 elliptic curve used in Bitcoin.

Schnorr signatures offer several advantages over the ECDSA signatures previously used in Bitcoin:

- **Provable security** — security proof under the discrete logarithm assumption
- **Linearity** — enables native multi-signature and key aggregation schemes (e.g. MuSig)
- **Batch verification** — multiple signatures can be verified faster together than individually
- **Smaller attack surface** — simpler construction compared to ECDSA

BIP-340 was activated on the Bitcoin mainnet as part of the Taproot soft fork (November 2021).
